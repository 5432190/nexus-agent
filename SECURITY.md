# Security Policy

## Supported Versions
| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | ✅                 |

## Reporting a Vulnerability
Email security@yourdomain.com. We respond within 48 hours.

## Key Management
- Signing keys are never committed to git
- KMS/HSM integration available in Enterprise tier
- Audit logs are immutable but never contain raw secrets
